¿Qué diferencias y semejanzas hay entre Java y JavaScript?

Diferencias:
  • Java es un lenguaje de programación orientado a objetos puros
    JavaScript se basa en prototipos y emula la programacion ortientada objetos
  • Java tiene variables definidas, JavaScript es un lenguaje mas fexible
  • Java fuertemente tipado, JavaScript debilmente tipado
Semejanzas
  • Provienen de C
  • Ambos trabajan POO

¿Qué métodos tiene el objeto Date? (Menciona al menos 5*)

  • getMonth() devuelve el numero del mes (inicia desde 0)
  • getDate() devuelve el dia del mes (inicia desde 0)
  • getDay() devuelve el dia de la semana (inicia desde 0)
  • getHours() devuelve la hora actual (inicia desde 0, formato 24hr)
  • getMinutes() devuelve el numero del mes (inicia desde 0)

¿Qué métodos tienen los arreglos? (Menciona al menos 5*)

  • lenght() regresa la longitud del arreglo
  • split() separa el arreglo por caractéres
  • push() agrega un valor al arreglo
  • pop() elimina un valor del arreglo
  • sort() ordena el arreglo

¿Cómo se declara una variable con alcance local dentro de una función?

Se declara con let

¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

Las variables globales son utilizadas, como su nombre lo dice, de manera global, es decir no existe ningun momento de muerte para estas, pueden ser utilizadas en cualquier parte del código
